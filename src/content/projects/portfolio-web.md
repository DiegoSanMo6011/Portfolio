## Objetivo
Construir un portafolio con **narrativa técnica sólida**, alto impacto visual y estructura escalable para nuevos proyectos.

## Enfoque técnico
- **Astro** con Content Collections para proyectos y certificados.
- **Tailwind CSS** para diseño consistente y componentes reutilizables.
- Arquitectura por secciones y layouts para crecimiento continuo.

## Aporte clave
Este sitio no es solo un escaparate: está diseñado para **vender proyectos complejos** con claridad, evidencia y jerarquía visual.

## Estado
Base lista para versión bilingüe y expansión de casos de estudio.
